Plateforme web connectant propriétaires de bornes de recharge et conducteurs de véhicules électriques.

Système complet avec :
- Authentification **JWT**
- Gestion de réservations
- Recherche géolocalisée avec **carte interactive**
- Tableaux de bord adaptés par rôle

Architecture **microservices** avec Spring Boot et React, utilisant MySQL pour les données relationnelles et MongoDB pour les données géospatiales.
