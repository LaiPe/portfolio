Drums Please

Site Web crée le 31/03/2024

Ce site est une maquette de site vitrine ayant pour objet une entreprise fictive de location de percussions. Il possède une page d'accueil qui présente l'entreprise et permet d'aiguiller l'utilisateur dans sa navigation sur le site. Il a des pages produits qui permettent à l'utilisateur d'accéder aux diverses informations des produits de manière fluide et dynamique. De plus, il intègre un espace utilisateur permettant de s'inscrire et de se connecter sur le site.

Ce site a de plus été conçu pour être consultable sur tous les supports : ordinateurs, tablettes, smartphones, etc...

De par sa nature de maquette et de démostration technique, ce site est hébergé de manière statique, ce qui veut dire qu’il ne traite aucune information qui lui est envoyé. Ainsi, bien qu’il y ait des formulaires de connexion et d’inscription, les informations qu’il contiendront s’ils sont remplis ne seront pas traitées.

Sokoban

Jeu Vidéo crée le 26/05/2023

Le jeu de Sokoban est un jeu de puzzle inventé au Japon dans les années 1980. Le joueur incarne un personnage qui doit pousser des caisses vers des emplacements spécifiques dans un entrepôt. L'objectif est de déplacer toutes les caisses sur les emplacements cibles pour terminer chaque niveau. Le joueur se déplace dans un environnement en 2D, représenté généralement par une grille. Il peut se déplacer horizontalement ou verticalement, mais il ne peut pas traverser les murs ni pousser les caisses de manière diagonale. Vous pourrez trouver de plus amples explications sur les règles du jeu de Sokoban dans le fichier « README.md » joint au dépôt du projet.

Réalisé dans le cadre de mes études, ce projet est une modélisation du jeu de Sokoban codée en C. Pour l'instant, le programme ne peut jouer qu'une partie de "test". Cependant, il n'est pas impossible que j'ajoute dans de futures mises à jour des modes de jeu plus élaborés. De plus, le projet contient également une rédaction disponible au format PDF détaillant les choix d’implémentations et de conception du programme. À noter que depuis la rédaction de ce "compte-rendu", le projet a connu des évolutions. Ainsi, certains points de la rédaction peuvent être désuets au regard de l'état actuel du projet. Cette rédaction est donc à replacer dans le contexte du projet tel qu'il était à sa version 0.1.0-alpha. Cette version est disponible ici ou bien en temps que realese sur le dépot Github.

Le fichier README contient, en plus des règles du jeu, la liste des commandes ainsi que les instructions relatives à la compilation et à l'exécution du programme.

La Réussite des Alliances

Jeu Vidéo crée le 24/04/2022

La Réussite est un jeu de cartes solitaire. Dans ce projet, les règles de la Réussite ont été simplifiées mais restent dans les grandes lignes les mêmes : regrouper les cartes en piles et essayer de posséder le moins de piles possibles. Ainsi, à chaque tour de jeu, le joueur pioche une carte qu’il dispose sur un banc de cartes, les unes à la suite des autres. Une carte peut former une pile si les cartes adjacentes possèdent la même valeur ou la même enseigne (carreau, trèfle, cœur ou pique). Si une des deux conditions est remplie, la carte sélectionnée pourra alors se poser sur la carte qui la précède. Vous pourrez trouver de plus amples explications sur les règles du jeu dans la rédaction jointe au projet.

Ce projet est donc la modélisation informatique de cette version de la Réussite. Réalisé dans le cadre de mes études, il est codé en Python et est accompagné par un compte-rendu complet détaillant l’implémentation, les choix de conceptions, les options et autres fonctionnalités contenu dans le projet.

De plus, est joint à ce projet un fichier README donnant les instructions nécessaires au démarrage du programme.
